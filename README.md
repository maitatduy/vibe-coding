# Vibe Coding

Repo học và thực hành **Vibe Coding** - ghi chú và làm project trong cùng một nơi.

## Mục tiêu

- Nắm vững tư duy **System Architect** thay vì chỉ biết gõ prompt cho AI
- Build 1 sản phẩm cá nhân hoàn chỉnh từ idea → deploy

## Cấu trúc repo

```
vibe-coding/
├── notes/          # Ghi chú Markdown theo từng module
├── playgrounds/    # Code thực hành nhỏ theo từng bài
└── project/        # Project chính
    ├── frontend/
    ├── backend/
    └── docs/       # PRD, schema, API contract
```
